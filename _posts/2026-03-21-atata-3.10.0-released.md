[Atata {{page.release_version}}](https://www.nuget.org/packages/Atata/{{page.release_version}})
is released with an improved StaleElementReferenceException handling and an upgrade of Selenium.WebDriver to v4.41.0.
{:.lead}

<!--more-->

## Changelog

### Changed

- Improve `StaleElementReferenceException` handling
- Upgrade Selenium.WebDriver package reference to v4.41.0
- Upgrade Atata.WebDriverExtras package reference to v3.6.0

## Summary

There was made a valuable improvement of `StaleElementReferenceException` handling, especially for control lists.
It was reproducible from time to time on some pages that refresh/update while an assertion of a control list items is executing.
Now it should become more stable.

The release contains an upgrade of Selenium.WebDriver package reference from v4.40.0 to v4.41.0.
Take a look at [Selenium.WebDriver CHANGELOG](https://github.com/SeleniumHQ/selenium/blob/trunk/dotnet/CHANGELOG)
to find out what was changed there.
