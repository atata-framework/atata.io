## Changelog

### Changed

- {% include issue.md id=858 type='mj' %} Upgrade Selenium.WebDriver package reference to v4.36.0
- {% include issue.md id=859 type='mj' %} Upgrade Atata.WebDriverExtras package reference to v3.3.0

## Summary

This is a minor update of Atata with an upgrade of Selenium.WebDriver package reference from v4.33.0 to v4.36.0.
Take a look at [Selenium.WebDriver CHANGELOG](https://github.com/SeleniumHQ/selenium/blob/trunk/dotnet/CHANGELOG)
to find out what was changed there.

Also Atata.WebDriverExtras package reference was upgraded from v3.2.0 to v3.3.0.
There was an improvement in `StaleSafely` class, see [Atata.WebDriverExtras Release 3.3.0](https://github.com/atata-framework/atata-webdriverextras/releases/tag/v3.3.0).
Now `StaleSafely.Execute` methods, in addition to `OpenQA.Selenium.StaleElementReferenceException`,
also handles `OpenQA.Selenium.UnknownErrorException` to make retries.
